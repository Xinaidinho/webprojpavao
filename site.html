Portfólio Do Aluno Eduardo Schneider

Neste site serão apresentadas atividades elaboradas pelo aluno de ciência da computação da PUC-PR, Eduardo Schneider, no primeiro semestre de sua gradução.

Experiência Criativa: Navegando na Computação

  • Na disciplina de experiência criativa, o maior objetivo era o desenvolvimento na área da programação utilizando a prática como modo de aprimoramento. A disciplina teve seu semestre voltado a três projetos, o primeiro sendo um jogo desenvolvido usando a plataforma Construct, o segundo um aplicativo multimídia utilizando o ambiente de desenvolvimento integrado Processing, e o terceiro um aplicativo Web utilizando HTML e CSS como principais ferramentas

  • Resolução de Problemas com Lógica Matemática

    Disciplinna com foco na melhora no pensamento lógico matemático voltado a programação, de maneira a trazer a capacidade de entender e resolver problemas do mundo real por meio da dedução matemática.

  • Fundamentos de Sistemas Ciberfísicos

    Disciplina onde foi lecionado os princíoopios fundamentais de conexão entre a hardware e software de um computador, onde contamos com aprendizados diversos sobre memórias processadores e outros componentes e finalizamos o semestre com um trabalho onde é apresentado uma aplicação utilizando o ESP32.

  • Raciocínio Algoríotimico

    Disciplina focada principalmente no desenvolvimento da habilidade de pensamento ligado a programação.

    Projeto I - Joquempô

    Projeto realizado em grupo visando o desenvolvimento de um jogo de joquempô em python.

    Projeto II - Batalha Naval

    Projeto realizado em grupo visando o desenvolvimento de um jogo de batalha naval em python utilizando matrizes. Projeto está em desenvolvimento
